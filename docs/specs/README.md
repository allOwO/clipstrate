## Clipstrate 是什么

macOS 26 原生剪贴板管理器，三个卖点：

1. **Liquid Glass 原生观感**——像内置 App；
2. **Maccy 级轻量**——常驻 < 30 MB、零泄露、零卡顿（硬指标）；
3. **「大爆炸」分词拆选**——复制的文本一键炸成词块，淘口令/手机号/邮箱等实体一键复制（macOS 市场空白）。

## 术语表

| 术语 | 含义 |
|---|---|
| **唤出面板 / SummonPanel** | 全局快捷键（默认 ⌥V）在光标/鼠标处弹出的横向卡片条，无外框、底边对齐（原型变体 C） |
| **分词层 / ChopOverlay** | 对一条文本"炸开"成词块的玻璃面板，含智能实体行 |
| **实体 / Entity** | 规则识别出的可一键复制片段：淘口令、手机号、邮箱、URL、验证码、快递单号… |
| **EntityHUD** | 复制后检测到实体时屏幕角落弹出的胶囊提示（点击/⌥X 展开分词层） |
| **Popover** | 点击菜单栏图标直接弹出的主界面（横向卡片流 + 设置/关于入口），不是菜单 |
| **堆栈 / Stack** | 连续复制多条后按顺序依次粘贴的模式（P1） |
| **通铺设置** | 设置窗口右侧一整页连续滚动 + 左侧锚点边栏 scroll-spy 高亮 |
| **接缝 / Seam** | 集中隔离 macOS 26-only API 的三个点（GlassSurface / PrivacyGate / 动画降级），业务代码不写 `#available` |

## 红线（任何实现不得违反）

1. 性能预算见 规划 §5.1，超标按 bug 处理；泄露清单见 规划 §5.3 与根目录 `AGENTS.md`。
2. 第三方依赖白名单：**GRDB.swift、KeyboardShortcuts**，新增依赖须用户批准。
3. 数据全本地（iCloud 备份除外且默认关闭前不联网）；尊重 ConcealedType；来源 App 获取不到就不显示，不猜。
4. macOS 26-only API 只能出现在三个接缝内。
