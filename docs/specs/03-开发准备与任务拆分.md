# 03 · 开发准备与任务拆分

## 1. 环境要求

- Apple Silicon Mac + macOS 26（Tahoe）+ Xcode 26（含 macOS 26 SDK）
- `brew install xcodegen`；（可选）`brew install swiftformat`
- 权限调试：开发构建每次重签会重置 TCC 授权，辅助功能需重新勾选——属正常现象。

## 2. 工程脚手架

仓库根目录创建 `project.yml`：

```yaml
name: Clipstrate
options:
  bundleIdPrefix: io.github.allowo
  deploymentTarget:
    macOS: "26.0"
packages:
  GRDB:
    url: https://github.com/groue/GRDB.swift
    from: 7.0.0
  KeyboardShortcuts:
    url: https://github.com/sindresorhus/KeyboardShortcuts
    from: 2.0.0
targets:
  Clipstrate:
    type: application
    platform: macOS
    sources: [Clipstrate]
    dependencies:
      - package: GRDB
      - package: KeyboardShortcuts
    settings:
      base:
        PRODUCT_BUNDLE_IDENTIFIER: io.github.allowo.clipstrate
        SWIFT_VERSION: "6.0"
        SWIFT_STRICT_CONCURRENCY: complete
        ARCHS: arm64
        MACOSX_DEPLOYMENT_TARGET: "26.0"
        ENABLE_HARDENED_RUNTIME: NO   # 发布(M4)改 YES 并公证
        INFOPLIST_KEY_LSUIElement: YES  # 无 Dock 图标
    info:
      path: Clipstrate/Info.plist
      properties:
        LSUIElement: true
        NSHumanReadableCopyright: "© 2026"
  ClipstrateTests:
    type: bundle.unit-test
    platform: macOS
    sources: [ClipstrateTests]
    dependencies:
      - target: Clipstrate
```

- **不开启 App Sandbox**（AX + CGEvent 需要）；不需要额外 entitlements 文件即可开发。
- 目录骨架按 02 §2 建：`Clipstrate/{App,Capture,Store,Chop,UI,System,Shared}/` + `ClipstrateTests/`。

常用命令：

```bash
xcodegen generate                                  # 生成 .xcodeproj（改 project.yml 后重跑）
xcodebuild -scheme Clipstrate -configuration Debug build
xcodebuild -scheme Clipstrate test                   # 单元测试
open Clipstrate.xcodeproj                            # 需要跑真 App 调权限时用 Xcode ⌘R
```

## 4. 给接手会话的执行约定

1. **开工前**：读 `AGENTS.md`（根目录）→ specs/README → 02 → 领任务对应的 01 章节；UI 任务同时打开 `prototype/index.html` 对照。
2. **顺序**：严格按 T 编号依赖推进；一个任务一个 commit（或 PR），commit message 引用任务号（如 `T1.3: 卡片条 UI`）。
3. **每任务完成即跑**：`xcodebuild test` + 该任务【验收】；M 级里程碑末尾跑 M3 性能/泄露闸门（M0/M1 期间只要求不肉眼可见劣化）。
4. **不确定就查 specs**；specs 没写的行为按"最像 macOS 内置 App 的做法"选择，并在 PR 描述里注明该决策。
5. **禁止**：新增依赖、把 `#available` 写出三接缝之外、跳过泄露清单（见 AGENTS.md）。

## 5. 遗留决策（已定，防反复）

- **需求已冻结**（2026-07-22 用户确认"这些是全部的需求"）：新想法一律进 backlog，不扩 v0.1 范围。
- 本地存储 = SQLite（GRDB）+ FTS5 trigram + blob 落盘；**不用** SwiftData/CoreData/Realm。
- iCloud = CloudDocs 文件夹写 `.clipstrate` 快照（Developer ID 无 iCloud entitlement）；CloudKit 留给 MAS 版，走 `BackupTransport` 接缝。
- 搜索入口 = Popover 常驻搜索框 + 唤出面板 type-to-search；范围 = 正文/文件名/图片 label/来源 App；图片 OCR 属 P2。
- 面板形态 = 原型变体 C（无框、底对齐、广告陈列感）；A/B 已废弃。
- 「粘贴」不做按钮；点击选中卡 = 粘贴。
- 设置 = 通铺 scroll-spy，不做分页式。
- 分词层保留玻璃容器（不无框化）。
- 来源 App 取不到就不显示。
- 默认快捷键：⌥V 唤出、⌥X 拆词。
